# Blueprint SDLC — Claude Code Plugin

> Turn Claude Code into a disciplined engineering team.

## Install

```
/plugin marketplace add skaisser/blueprint-plugin
/plugin install blueprint
```

That's it. 27 skills, audit hooks, MCP servers, and status line — all active instantly.

## What's Included

- **27 slash commands** covering the full SDLC pipeline
- **15-rule audit hook** enforcing code quality on every tool call
- **MCP servers**: Context7 (live docs) + Sequential Thinking
- **Status line** with context usage, burn rate, and git info
- **Templates** for git hooks, GitHub Actions, and project scaffold

## CLI Binary (Optional)

The audit hook requires the Blueprint CLI binary. It auto-installs on first use, or install manually:

```bash
brew tap skaisser/tap && brew install blueprint
```
